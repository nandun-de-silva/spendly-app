DFD — Level 1 (System Decomposition)

6 core processes, data stores, and all major data flows within Spendly

👤 User
🔐 Biometric Sensor
📱 AsyncStorage
📁 Downloads Folder
1
Authenticate User
Validates PIN or biometric. Grants or denies access. Stores encrypted PIN via Expo SecureStore.
► PIN / Biometric► Stored PIN hash◀ Auth granted / denied
2
Manage Transactions
Creates, reads, updates, and deletes income and expense records. Validates input, triggers balance recalculation.
► Transaction data◀ Transaction list◀ Updated balance
3
Calculate Dashboard
Aggregates transactions by month. Calculates totals, pie chart data, bar chart data, and trend line points.
► Transaction records◀ Chart data◀ Monthly summary
4
Manage Budgets & Alerts
Stores monthly budgets per category. Compares spending against limits. Fires 80% and 100% exceeded alerts.
► Budget limit► Category spending◀ Alert notification
5
Filter & Search
Applies user-defined filters (type, category, date range) and keyword search to the transaction list in real time.
► Filter criteria► Search keyword◀ Filtered list
6
Export Data
Reads transactions from AsyncStorage, applies optional date filter, generates CSV, writes to device Downloads folder.
► Export request► Date range filter◀ CSV file
